- Python 3+ instalado, lo puede descargar de: https://www.python.org/downloads/windows/
Es importante que durante la instalación marque la casilla para agregar python a las variables de entorno PATH, de lo contrario, tendrá que hacerlo de manera manual.

- Una vez instalado python, correr los siguientes comandos en el cmd para instalar las librerías:
	- pip install --upgrade pip
	- pip install beautifulsoup4
	- pip install bs4
	- pip install selenium
	- pip install request

- Tambien debe instalar el navegador firefox y posicionar el archivo "geckodriver.exe" en alguna carpeta a discresión suya, y agregar la ruta de este archivo a la variable de entorno PATH. No es necesario que corra el archivo .exe
